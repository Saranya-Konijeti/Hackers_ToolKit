Nmap

Network exploration tool and security scanner

Overview

Network discovery, port scanning, OS detection.
  1. Host discovery: Identify the hosts on the network.
  2. Port Scanning: Find open ports on target systems.
  3. Service Detection: Determine run service, versions.
  4. OS Fingerprinting: Identify target operating system.

Common Commands

  1. $ nmap -sn 192.168.1.0/24
  2. $ nmap -sV -sC target.com
  3. $ nmap -sS 192.168.25.36

Advanced usage and best cases

Custom script automation and integration.

Wireshark

Network protocol analyzer for packet inspection

Overview

Traffic nalysis, protocol debugging, network troubleshooting.
  1. Capture: Record and analyze network traffic in real time.
  2. Filter: Apply display filters to focus on specific traffic.
  3. Analyze: Examine packet contents and protocol details.

Common Commands

  1. $ tshark -i eth0 -f "tcp port 80
  2. $ tshark -r capture.pcap -Y "http"
  3. $ tshark -i any -w output.pcap

Burpsuite

Web application security testing platform

Overview

Web app testing, vulnerability scanning, API security
  1. Proxy: Intercept and modify HTTP requests
  2. Spider: Crawl the application to map content.
  3. Intruder: Perform automated attacks with payloads.

Common Commands

  1. $ burpsuite --headless --project-file=scan.
  2. $ burpsuite --license-file=/path/to/license.txt
  3. $ burpsuite --scan --url=https://target.com
