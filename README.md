# Hacker’s Toolkit

Hacker’s Toolkit is a beginner-friendly cybersecurity learning website that explains commonly used ethical hacking tools such as Nmap, Wireshark, Burp Suite, and Metasploit.

It is built using HTML, CSS, and JavaScript and includes an AI-powered chatbot to answer questions related to cybersecurity tools.

## Features

- Detailed overview of popular security tools  
- Common commands and use cases  
- Advanced usage and best practices  
- AI Chat Assistant (Gemini API integration)  
- Cyber-themed UI  
- Responsive design  

## Tools Covered

- Nmap – Network scanning and discovery  
- Wireshark – Packet capture and analysis  
- Burp Suite – Web application security testing  
- Metasploit – Penetration testing framework  

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Google Gemini API (Client-side integration)
